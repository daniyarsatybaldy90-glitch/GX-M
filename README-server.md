# GX-M400 Backend

Минимальный сервер для приёма заявок и раздачи собранного фронтенда.

## Локальный запуск

```
npm i
npm i express cors
npm run build
node server/index.js
```

После запуска приложение доступно на http://localhost:3000.

## Публикация временной внешней ссылки

```
npx localtunnel --port 3000
```

Команда выдаст HTTPS-ссылку, которую можно передать клиенту.
