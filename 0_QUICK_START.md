## 🔐 КЛЮЧЕВАЯ ИНФОРМАЦИЯ

### Google Client ID
```
Где получить:  Google Cloud Console → OAuth 2.0 Client ID
Где добавить:  Vercel → Environment Variables → VITE_GOOGLE_CLIENT_ID
❌ НЕ кладите в исходный код!
```

### Переменные окружения
```
Локально: cp .env.example .env.local
На Vercel: Project Settings → Environment Variables
```

### GitHub Personal Access Token
```
Где создать: GitHub Settings → Personal Access Tokens
Используйте вместо пароля при git push
❌ Не делитесь токеном!
```
