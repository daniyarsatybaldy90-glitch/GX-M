🔐 ВАЖНО ПОМНИТЬ
═══════════════════════════════════════════════════════════════

❗ Google Client ID
   → Получите в Google Cloud Console
   → Добавьте в Vercel Environment Variables
   → НЕ кладите в исходный код

❗ Personal Access Token для GitHub
   → Создайте в GitHub Settings
   → Используйте вместо пароля при git push
   → Не делитесь им!

❗ Vercel редирект
   → Если используете свой домен
   → Добавьте его в Google OAuth settings
